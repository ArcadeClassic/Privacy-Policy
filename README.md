# ArcadeClassic Network. Privacy Policy
Last Edited. 12/17/25

## 1. Introduction

Welcome to ArcadeClassic. A Minecraft multiplayer network operated by the ArcadeClassic team. This Privacy Policy explains how player data is collected, used, stored, and protected when you play on ArcadeClassic servers or use related services such as our website or Discord. By using ArcadeClassic services, you agree to this policy. If you do not agree, do not use our services.

## 2. Information We Collect

### 2.1 Data Collected

When you play on ArcadeClassic, we collect limited data required to operate the network, including:
- Minecraft username and UUID
- IP address for security and moderation
- In game statistics, progress, and inventories
- Punishment and moderation history
- Discord user ID if your account is linked

## 3. How We Use Your Information

We use collected data to:
- Run and maintain Minecraft servers
- Save player progress and stats
- Enforce rules and prevent cheating or abuse
- Improve server performance and gameplay
- Provide support and resolve issues

## 4. Data Disclosure

### 4.1 Legal Compliance

Player data may be disclosed if required by law or valid legal requests from authorities.

### 4.2 Service Providers

Data may be shared with trusted services used for hosting, security, analytics, or backups. Only required data is shared.

## 5. Data Storage and Security

### 5.1 Data Storage

Data is stored on secured servers using standard industry protections to reduce unauthorized access, loss, or misuse.

### 5.2 Data Retention

Data is kept only as long as needed for server operation, moderation, or legal reasons. If you stop playing or request deletion, eligible data is removed unless retention is required.

### 5.3 Security Measures

We use administrative and technical safeguards to protect data. No system guarantees complete security.
